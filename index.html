Proyecto de Inteligencia Artificial

Clasificación de Condiciones de Desbalance mediante Machine Learning y Análisis de Vibraciones

Este proyecto tiene como objetivo clasificar automáticamente distintas condiciones de desbalance en un sistema rotativo mediante el análisis de señales de vibración. Para ello se adquirieron datos experimentales utilizando acelerómetros, se extrajeron características estadísticas y espectrales de las señales, y posteriormente se entrenaron modelos de Machine Learning para identificar el estado del sistema.

  • Dataset: Datos obtenidos de un banco experimental rotativo bajo cuatro condiciones: Normal, Unbalance 19 g, Unbalance 23 g y Unbalance 27 g.
  • Características: RMS, Peak, Crest Factor, Kurtosis, Desviación Estándar, Media Absoluta, Frecuencia Dominante, Amplitud Dominante, Energía Espectral, Centroide Espectral y Bandwidth Espectral.
  • Modelo: Se evaluaron Random Forest y SVM. El modelo Random Forest presentó el mejor desempeño y fue seleccionado como modelo final.
  • Visualización: Se realizaron gráficos temporales, FFT, matrices de confusión y comparación de métricas para evaluar el desempeño del modelo.
  • Repositorio: github.com/fernandoBenitezFIUNA/proyecto_vibraciones
